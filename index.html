Advanced Audio Visual Processing Final Project

This is the final project for Goldsmiths College, Advanced Audio Visual Processing. It is a webcam image filter that uses circles to represent pixels, and combines all the circles to create an image. Using p5.js we iterate through the image array and apply a given fill based on the Red, Green, and Blue values of the current pixel. There are three modes for the image: Color, Sepia, and Black and White. Depending on which mode is selected, the image is rendered using a pixel-based image filter algorithm. The color mode engages the three sliders below the image, and allows you to proportionally filter the pixels based on the red, green, and blue multipliers. Sepia mode uses an pixel-based color conversion to go from full color to hues of brown. Black and White mode takes the luminosity of each pixel and uses that for the single-channel pixel value.
